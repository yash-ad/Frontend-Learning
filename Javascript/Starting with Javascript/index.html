Switch-Statement

In javascript switch statement is used for decision making.The switch statement evaluates an expression and executes the corrresponding body that matches the expressions result.

[Syntax] switch(Variable/Expression) { case value 1 : //Body of case 1 break; case value 2 : //Body of case 2 break; case value 3 : //Body of value 3 break; case value so on : //Body of case value So on break; default : //Body of default }

Important Notes

The switch statement is used to perform different actions based on diferrent conditions.

  1. Use switch to specify many alternative block of code to be executed.
  2. The switch expression is evaluated once.
  3. The value of the expression is compared with the values of each case.
  4. If there is a match , the associated block of code is executed.
  5. If there is no match, the default code block is executed.

The break keyword:-

When javascript reaches a break keyword, it breaks out of the switch block.this will stop the execution inside the switch block.it is not necessary to break the last case in a switch block.the block ends there anyway. dont forget to add a break at the very end of each case in a switch statement

The default keyword:-

The default keyword specifies the code to run if there is no case match. if default is not the lastcase in the switch block,REMEMBER to end the default case with a break.

The break statement is optional.if the break statement is encountered,the switch statement ends. if the the break statement is not used,the casesafter the matching case are also executed. the default clause is also an optional. **Multiple cases in switch statements:-In a javascript switch statement,cases can be grouped to share the same code.

What are loops in javascript?

In programming language , loops are used to repeat a block of code under certain condition.For example if you want to show a message 100 times,then you can use a loop.

Javascript developers use loops to continually execute and repeated blocks of code until a caetain condition is satisfied.

  • Loops can execute a block of code a number of times.
  • Loops are handly,if you want to run a code the same over and over again , each time with a diferrent value.

    • Different Kinds of loops:-

    Javascript supports different kinds of loops.

    1. for:- Loops through a block of code a number of times.
    2. for/In:-Loops through the properties of an object.
    3. for/of:-Loops through the values of an iterable object.
    4. Whileloop:-Loops through a block of code while a specified condition is True.
    5. doWhileloop:- Also loops through a block of code while a specified condition is True.

    The for-Loop

    for is a declaration keyword,and the counter is a variable which is typically named i for iterator.The for statements creates a loop with 3 optional terms or expressions.

    for(Intialexpression(Intialization) ; Condition ; update expression(Iteration)) { //forLoop body code to be executed; }
