A pseudo class is used to define a special state or selectors of an element

Examples are below:>

  1. Visited
  2. Link
  3. hover
  4. active
  5. first-child
  6. last-child
  7. nth-child

A css pseudo element is used to style specified parts of an element.

Examples are below:>

  1. ::First-letter
  2. ::First-line
  3. ::Before
  4. ::After
